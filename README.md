**AI-Powered Code Review Assistant** — Your Personal Senior Engineer

## ✨ Features
- Paste any code → Get instant AI review
- Detects bugs, security issues, performance problems
- Suggests improvements with explanations
- Supports 10+ programming languages
- Beautiful dark UI with Tailwind + shadcn
- Fast response using Groq (Llama 3.3 70B)

## 🛠 Tech Stack
- **Frontend**: Next.js 15 (App Router) + TypeScript + Tailwind CSS
- **Backend**: FastAPI (Python)
- **AI Model**: Groq + Llama 3.3 70B
- **Deployment Ready**: Vercel (frontend) + Render (backend)
